# Specification: FindashLayout.vue (파이낸셜 대시보드 레이아웃)

## Overview (개요)
A wrapper layout component for the Financial Dashboard (Findash) module.
파이낸셜 대시보드(Findash) 모듈을 위한 래퍼(Wrapper) 레이아웃 컴포넌트입니다.

## Components (구성 요소)
- **Router View**: Container for nested routes within the Findash module. (Findash 모듈 내의 중첩된 라우트들이 표시되는 컨테이너)

## Functional Analysis (기능 분석)
- **Working (정상 동작)**: Properly routing to child components (Index, Global Assets, Market Lab, Sector Comparison). (하위 페이지들로의 라우팅 정상 동작)
