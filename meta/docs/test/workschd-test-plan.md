# Workschd 모듈 테스트 계획

**버전**: 1.0  
**작성일**: 2026-01-18  
**모듈**: `frontend/src/views/workschd`, `backend/src/modules/workschd`

---

## 개요

본 문서는 **Workschd** (근무 일정 관리) 모듈의 통합 테스트 계획을 기술합니다:
- 인증 및 사용자 관리
- 팀 관리 (등록, 멤버, 매장)
- 업무 관리 (CRUD, 캘린더, 템플릿)
- 근로자 플로우 (참여 신청, 출퇴근)
- 모바일 뷰
- 알림 및 통계

---

## 테스트 흐름도

```mermaid
flowchart TB
    subgraph Auth["인증"]
        A1[로그인] --> A2[OAuth]
        A2 --> A3[회원가입]
        A3 --> A4[로그아웃]
    end

    subgraph Team["팀 관리"]
        T1[팀 등록] --> T2[팀 조회]
        T2 --> T3[팀 상세]
        T3 --> T4[멤버 탭]
        T4 --> T5[매장 탭]
        T5 --> T6[일정 설정]
        T3 --> T7[초대 링크]
    end

    subgraph Task["업무 관리"]
        TK1[업무 생성] --> TK2[업무 그리드]
        TK2 --> TK3[캘린더 뷰]
        TK3 --> TK4[드래그 &#38; 드롭]
        TK4 --> TK5[시프트 템플릿]
    end

    subgraph Worker["근로자 플로우"]
        W1[참여 신청] --> W2[승인/거절]

    end

    subgraph Mobile["모바일 뷰"]
        M1[업무 목록 모바일]

    end
```
