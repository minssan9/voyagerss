# Workschd Module

This module handles work scheduling, team management, and task tracking.

## Components & Views

- **Auth**: Authentication flows (Login/Register).
- **Task**: Task management interfaces.
- **Team**: Team management, member lists, or roles.
- **Account**: User account settings.
- **Admin**: Administrative controls.
- **Main**: Core application views for the authenticated user.

## Structure
- Located in `src/views/workschd/`
- Highly structured with subdirectories for each domain area (`auth`, `task`, `team`, etc.).
- `agents.md` present in source, possibly describing specific agent behaviors for this module.
